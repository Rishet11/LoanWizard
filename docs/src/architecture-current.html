Shared contracts

packages/contracts

Frozen TypeScript types, Zod schemas and mock data. Every stream consumes the same session, perception and decision shapes, so the three streams integrate without drift.

Build model

Three parallel streams (A/B/C) against the frozen contract package — independent development, typed integration.

Deploy target

Two Docker Spaces (web + ML) on Hugging Face, Neon Postgres for the shared audit trail.

Explainability

Every offer ships reason codes + narrative; the input snapshot is frozen so the decision replays on the same model version.
